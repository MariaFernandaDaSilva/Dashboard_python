# Dashboard_python

Este projeto em Python realiza uma **análise básica de vendas** e gera uma **visualização gráfica** do desempenho de dois serviços ao longo de seis meses. É uma demonstração prática do uso de **Pandas** para manipulação de dados e **Matplotlib** para visualização.

---

## Funcionalidades

- Criação de um **DataFrame** a partir de dados brutos (dicionário Python).
- Cálculo do **total de vendas por mês** somando dois serviços diferentes.
- Geração de **estatísticas básicas** (média e valor máximo) das vendas.
- Criação de um **gráfico de barras** mostrando o total de vendas mensais com valores em destaque.

---

## Tecnologias Utilizadas

- **Python 3**
- **Pandas**: manipulação e análise de dados.
- **Matplotlib**: criação de gráficos e visualizações.
